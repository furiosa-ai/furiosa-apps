# LLM Assistant
These examples contain AI assistant applications that use Furiosa-LLM as the inference backend.

## Applications

| Application | Description | Interface | Category |
|---|---|---|---|
| [OpenClaw](/llm-assistant/openclaw) |  A setup script that adds Furiosa RNGD as a backend provider to OpenClaw, an open-source AI agent platform with chat channels, memory, and tool use. | Web App / Agent Platform | Personal AI Assistant |
| [Weekly News Reporter](/llm-assistant/weekly-news-reporter) | A multi-agent system built with **LangGraph** that curates, summarizes, and compiles AI/ML news into a structured weekly report. | Agent Workflow | Multi-Agent Orchestration |
| [Financial Analysis](/llm-assistant/financial-analysis) | A real-time pipeline that streams stock-market tweets and classifies their sentiment using an LLM with live system metrics. | Streaming Dashboard | Real-time Inference Monitoring |
