# Coding Agent
These references contain AI-powered coding tools that use Furiosa-LLM as the inference backend.

## Applications

| Application | Description | Interface | Category |
|---|---|---|---|
| [OpenCode](/coding-agent/opencode) | Integrates `furiosa-llm` with the open-source coding agent **OpenCode** to enable accelerated code generation and developer assistance. | CLI / Agent | Code Assistant |
| [Unit Test Generator](/coding-agent/unit-test-generator) | A Streamlit interface that generates code explanations and `unittest` test cases from uploaded Python source files. | Streamlit Web App | Automated Test Generation |
